Hotel Management System

This was Application is developmed for mainaging a Hotel related thing like Billing System, Room Allocated, Salary of Employee.

This provide multiple functionality where you can generate a Bill of any user or we can check if rooms are available in Hotel or not for allocation. We can also mark a attendance of employees according to which we can manage a salary of employee also.

The whole project i have create in Swing's Java Framework and in AWT. Where i have created a different UI Components which is used to make a interaction with users and also I have used a Database(JDBC) which is userd to store the data of user's.

The Database which i have used is MySql(MariaDB)

Technologies:
  • - Core Java
  • - MySql
  • - Swings Framework
